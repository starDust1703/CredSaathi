# Cred Saathi

## This is a banking related, AI-agentic loan approval app

### Techs Used:
1. Next.js
2. Tailwind CSS

### Getting Started:

On your terminal, run the following commands:
```bash
npm i

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
